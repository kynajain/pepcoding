Keyboard Shortcuts

Operation Icon Keyboard Shortcut Description
Play/Pause Spacebar To play or pause the video.
Forward P Forward video by 5 seconds.
Rewind p Rewind video by 5 seconds.
Playback Rate * Increase playback rate of the video [1, 1.5 or 2 times the original].
Volume + Volume slider V Increase the volume of video.
Volume - Volume slider v Decrease the volume of video.
Fullscreen f Toggles full-screen mode.
Size + Size bar S Relevant only in split or pip view. Increases size of code screen in split view, and inner picture in pip view.
Size - Size bar s Relevant only in split or pip view. Decreases size of code screen in split view, and inner picture in pip view.
Board-only view c and C Brings in full view with board only.
Code-only view c and C Brings in full view with code only.
Split view c and C Brings in split view with shared focus on board and code.
Pip Code-First view c and C Brings in picture-in-picture view with focus on code.
Pip Board-First view c and C Brings in picture-in-picture view with focus on board.
Direction d Changes location of inner picture in pip views.
Help h Opens up the help menu.
